# JSON/YAML Schema Validator Tool

## Description
A tool that allows users to paste JSON or YAML content and a corresponding schema to validate structure and types.

## Functionality
- Input field for data (JSON/YAML)
- Input field for schema (JSON Schema)
- Validation button
- Error/Success display
- Toggle between JSON and YAML mode

## Implementation Plan
1. Create directory `json-schema-validator/`.
2. Add `index.html` (UI).
3. Add `script.js` (Validation logic using a library or standard JS).
4. Update `index.html` (main navigation) to include link.
